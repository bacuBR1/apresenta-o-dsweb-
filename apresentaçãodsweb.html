Media Queries

As media queries são recursos do CSS que permitem aplicar estilos diferentes dependendo do tamanho da tela, resolução ou até orientação do dispositivo. Em outras palavras: elas fazem seu site ficar responsivo, ou seja, bonito tanto no celular quanto no computador.

Como funciona?

Você define uma condição. Se ela for verdadeira, o CSS dentro dela é aplicado. ex: @media(max-width: 1360px){ body{ background-color: cornFlowerBlue; } }

Tipos mais usados

max-width

Usado para determinar até qual tamanho máximo da tela a função dentro do @media irá ser executada. ex.: @media (max-width: 600px) { body { background-color: black; } } /*O fundo da tela até 600px irá ficar preta*/

min-width

Usado para determinar o tamanho mínimo que a função dentro do @media irá ser executada. ex.: @media (min-width: 600px) { body { background-color: black; } } /*O fundo da tela de no mínimo 600px irá ficar preta*/

OBS

Também é possível alternar os dois, por exemplo: @media (min-width: 600px) and (max-width: 1360px) { body { background-color: black; } }

<div>

@media

é a função que se usa no media queries ex: @media (min-width: 600px) { body { background-color: black; } }

Breakpoints

Breakpoints são os pontos de largura da tela onde o layout “quebra” e precisa mudar. /*São os valores que você usa nas media queries para adaptar o design.*/ Exemplo: @media (max-width: 600px) { .container { flex-direction: column; } } /*O breakpoint é justamente o 600px, é onde os objetos do site iriam “quebrar” a tela, nesse caso os blocos do container iriam ficar em coluna, isso ajuda muito na mudança de telas grandes para telas pequenas*/

Mobile First

Mobile-first É uma forma de desenvolver o CSS onde se começa pelo celular e vai adaptando a telas maiores, justamente porque hoje a maioria dos acessos vem de celulares.
