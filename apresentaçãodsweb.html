                <div class="mediaqueries">

                <p>Media Queries</p>
                <p>
                    As media queries são recursos do CSS que permitem aplicar estilos diferentes dependendo do tamanho da tela, resolução ou até orientação do dispositivo. 
                    Em outras palavras: elas fazem seu site ficar responsivo, ou seja, bonito tanto no celular quanto no computador.
                </p>

                <p>Como funciona?</p>
                <p>
                    Você define uma condição. Se ela for verdadeira, o CSS dentro dela é aplicado.
                    ex:
                    @media(max-width: 1360px){
                        body{
                            background-color: cornFlowerBlue;
                        }
                    }
                </p>

                <p>Tipos mais usados</p>

                <p>max-width</p>
                <p>Usado para determinar até qual tamanho máximo da tela a função dentro do @media irá ser executada.</p>
                <p>
                    @media (max-width: 600px) {
                        body {
                            background-color: black;
                        }
                    }
                </p>

                <p>min-width</p>
                <p>Usado para determinar o tamanho mínimo que a função dentro do @media irá ser executada</p>
                <p>
                    @media (min-width: 600px) {
                        body {
                            background-color: black;
                        }
                    }
                </p>

                <p>OBS</p>
                <p>
                Também é possível alternar os dois, por exemplo:
                    @media (min-width: 600px) and (max-width: 1360px) {
                        body {
                            background-color: black;
                        }
                    }
                </p>

                <div>
            </div>

            <div class="media">
                <p>@media</p>
                <p>
                é a função que se usa no media queries ex:
                    @media (min-width: 600px) {
                        body {
                            background-color: black;
                        }
                    }
                </p>
            </div>

            <div class="breakpoints">
                <p>Breakpoints</p>
                <p>
                    Breakpoints são os pontos de largura da tela onde o layout “quebra” e precisa mudar.
                    /*São os valores que você usa nas media queries para adaptar o design.*/</p>
                <p>
                    @media (max-width: 600px) {
                        .container {
                            flex-direction: column;
                        }
                    }
                </p>
